Glossary

Confused about all the terms and acronyms? Take a look below and see if you can find the term you're confused about. See the home page for contact info if you're still confused about anything.

Northeastern Electric Racing (aka NER)
Northeastern's official electric race car building club, building an all-electric Formula One-style race car. NER is a club based in the college of engineering with opprotunties for students to learn business, mechanical engineering, and electrical engineering skills.
Project Management (aka PM)
Project management is the practice of organizing the work of a team and lead them to success in a certain period of time. The primary challenge of project management is to achieve all of the project goals within the given constraints of scope, time, quality, and budget.
Gantt Chart
A timeline view of a project that illustrates the squence of project phases (often equivalent to work packages) and their durations.
Work Package (aka WP)
A work package is a well-defined unit within a project. Work packages within NER include a unique identifying number (see WBS#),
Change Request
A change request is a request made by project leads to request a change to be made to a specific project. Change requests are primarily focused on changes being made to the project's given constraints (see PM). Sufficient documentation is requested to acompany the change request in order to better document changes and offer explanation for the changes.
Work Breakdown Structure Number (aka WBS#)
A work breakdown structure number is a unique identifying number assigned to each element of the work breakdown structure. The WBS# allows easy reference to a specific project component.
